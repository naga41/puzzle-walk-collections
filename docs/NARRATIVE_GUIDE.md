# Narrative Guidelines (世界観と執筆スキル)

「謎解き街歩きコレクション」は、単なるクイズゲームではありません。
都市の歴史や失われた記憶を辿る**「思索的な体験」**を提供する文学的ミステリーです。
今後AIージェントが自律的に新しいシナリオやコンテンツを生成する場合は、必ず以下の「スキル」と「世界観」を適用してください。

## 1. 原則的なトーン＆マナー
- **静謐さ（Quietness）**: 派手な演出や言葉を避け、静かで落ち着いた言葉選びをする。
- **時間の蓄積（Passage of Time）**: 現在の風景の下に、過去の風景が重なっているような「地層的」な視点を強調する。
- **水と記憶**: 「水」「海」「川」「水没」「記憶」「アーカイブ」「幻」「輪郭」といったモチーフを積極的に用いる。

## 2. NG表現と推奨表現のリスト
AIのデフォルトである「アシスタント的な言葉遣い」「サイバー/ハッカー的な言葉遣い」は強く禁止されます。

| ❌ NG (サイバー/システム的) | ◯ 推奨 (思索的/文学的) |
|---|---|
| システムを起動 | 記憶を辿る |
| インターフェース / OS | 都市の輪郭 / 街の鼓動 |
| エラー / パスワード違い | ピントが合わない / まだ焦点が結ばれない |
| 正解です！ / ロック解除 | 解像度が上がった / 記憶が繋がった |
| ミッション / ゲーム | 観測 / 追体験 |

## 3. キャラクター設定: 建築家「葛城（かつらぎ）」
（※第1弾『東京再発見アーカイブ』および今後のシリーズに登場するキーパーソン）
- **時代**: 1923年（関東大震災の直前）に活躍した若き建築家。
- **思想**: 近代化で均質化されていく都市を憂い、江戸時代の名残や「水」との共生を追求した理想の『水都の設計図』を描いていた。
- **テイスト**: 葛城の独白（`monologue`）は、詩的でありながら、都市の成り立ちに対する鋭い観察眼を持っている。感情的になりすぎず、冷徹な観察記録のように書くこと。

## 4. プロップ（小道具）画像のAI生成プロンプトのコツ
シナリオ中に挟む画像（`imageUrl`）をAI画像生成で作成する場合、ゲームアセットのようなポップなものは浮いてしまいます。以下のキーワードをプロンプトに入れることで、古い手記に挟まれたアイテムの質感を表現してください。
- `A blurry, faded minimal sepia ink sketch drawn on old parchment paper` (古い羊皮紙に描かれたセピアのスケッチ)
- `A grainy, faded monochrome early 1900s photograph, vintage style` (ノイズの乗った1900年代の古いモノクロ写真)
- `cyanotype style, technical drawing mixed with dreamy watercolor` (青焼き・ブループリント風の夢のような図面)
- `No background frames. Elegant, lots of negative space.` (背景枠なし、余白を多く)
